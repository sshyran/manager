## Install

```sh
yarn add @ovh-ux/manager-telecom-dashboard
```

## Usage

```js
import angular from 'angular';
import ovhManagerTelecomDashboard from '@ovh-ux/manager-telecom-dashboard';

angular
  .module('myApp', [
    ovhManagerTelecomDashboard,
  ]);
```
